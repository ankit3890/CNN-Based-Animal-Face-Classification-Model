Model Specifications

Details of the underlying PyTorch Convolutional Neural Network (CNN).

Hyperparameters

  • Optimizer: Adam
  • Learning Rate: 1e-4 (0.0001)
  • Batch Size: 16
  • Loss Function: CrossEntropyLoss
  • Training Device: CUDA (NVIDIA GPU)

Network Architecture

Input Tensor (3 Channels, 128x128 pixels)
↓
Convolution Block 1 Conv2d (3 ➔ 32) | MaxPool2d (2x2) | ReLU
↓
Convolution Block 2 Conv2d (32 ➔ 64) | MaxPool2d (2x2) | ReLU
↓
Convolution Block 3 Conv2d (64 ➔ 128) | MaxPool2d (2x2) | ReLU
↓
Flatten 128 * 16 * 16 = 32,768 units
↓
Fully Connected Layer Linear (32,768 ➔ 128) | ReLU
↓
Output Classifier Linear (128 ➔ 3 Logits)

Built with PyTorch, FastAPI, and Vanilla JavaScript. Dataset: AFHQ (Animal Faces High Quality).
